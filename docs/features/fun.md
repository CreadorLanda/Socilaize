## 🔔 Nudges

### O que são Nudges?

Uma notificação "poke" que chama atenção de forma divertida:

### Tipos de Nudge

| Emoji | Nome | Mensagem |
|-------|------|---------|
| 👋 | Oi | " está te dando um olá! 👋" |
| 💨 | Ei | " está te mostrando! 💨" |
| 😱 | Uau | " ficou impressionado! 😱" |
| 🥰 | Oi amor | " quer dizer olá! 🥰" |
| 🎉 | Yay | " está animado! 🎉" |

### Enviar Nudge

1. Abra o chat
2. Toque no ✉️ → Nudge
3. Selecione tipo

### Por Comando

```
/nudge
/nudge wave
/nudge poke
/nudge hey
```

## 🖼️ Display Pictures (DP)

### O que são Display Pictures?

Imagen de perfil que aparece ao lado do seu имя, como no MSN:

### Tipos de DP

- Imagem estática
- Animated GIF
- Video short

### Configurar

1. Perfil → Display Picture
2. Selecione imagem
3. Defina visibility

### Categorias

Organize suas DPs:
- Work
- Fun
- Family
- Events

# 📜 Política de Privacidade

> Política de Privacidade do Socialize.

---

## Visão Geral

Esta política descreve como coletamos, usamos, armazenamos e protegemos seus dados pessoais.

---

## 📋 Dados Coletados

### Dados Necessários

| Dado | Por quê | Retenção |
|------|---------|---------|
| **Número de telefone** | Identificação | Até exclusão |
| **Mensagens** | Comunicação | Até exclusão |
| **Contatos** | Conexões | Até exclusão |
| **Perfil** | Identidade | Até exclusão |

### Dados Opcionais

| Dado | Por quê | Retenção |
|------|---------|---------|
| **Biometria** | Auth | Até exclusão |
| **Localização** | Status | Até exclusão |
| **Dados de uso** | Melhorias | 90 dias |

### Dados que NÃO Coletamos

```
❌ contacts from device
❌ microphone without consent
❌ precise location
❌ clipboard content
❌ other apps data
```

---

## 🔒 Como Protegemos Seus Dados

### Criptografia

| Dado | Em Trânsito | Em Repouso |
|------|-------------|------------|
| Mensagens | ✅ E2E | ✅ AES-256 |
| Arquivos | ✅ TLS | ✅ AES-256 |
| Keys | - | ✅ Secure Enclave |

### Acesso

- Apenas dados mínimos necessários
- Autenticação fuerte
- Logs de auditoria
- 2FA disponível

---

## 🌍 Transferência de Dados

### Suas Opções

| Opção | Descrição |
|-------|-----------|
| Self-host | Você controla tudo |
| Cloud | Servidores nossos |
| Local | Apenas no device |

### Cross-Border

Seus dados podem ser processados em:
- Brasil (padrão)
- EUA
- Europa

Escolha em: Configurações → Privacidade → Localização

## ⏱️ Retenção de Dados

### Políticas de Retenção

| Tipo de Dado | Período | Notas |
|--------------|--------|-------|
| Mensagens | Você escolhe | 30d, 1y, forever |
| Mídia | Você escolhe | |
| Session logs | 30 dias | |
| Analytics | 90 dias | Agregado |
| Backups | Até delete | Criptografado |

---

## ❌ Seus Direitos

### GDPR / LGPD Rights

Você tem direito de:

| Direito | Descrição |
|---------|-----------|
| **Acesso** | Ver seus dados |
| **Correção** | Corrigir dados |
| **Exclusão** | Deletar dados |
| **Portabilidade** | Exportar dados |
| **Objecção** | Recusar processamento |
| **Revogação** | Retirar consentimento |

### Como Exercer

```
📧 privacidade@socialize.app
🗑️ Settings → Privacidade → Minhas Solicitações
📬 Resposta em até 72 horas
```

---

## 🔐 Crianças

O Socialize é para maiores de 13 anos.

Se percebermos dados de menores, excluímos imediatamente.

# ❓ FAQ - Frequently Asked Questions

> Answers to common questions about Socialize.

---

## 📋 General

### What is Socialize?

Socialize is an open-source, highly customizable, privacy-focused messenger. It's inspired by the customization freedom users love in chat mods, but built from scratch with security and innovation.

### Is Socialize free?

Yes! Socialize is completely open-source and free. You can self-host or use the managed version when available.

### What platforms are supported?

- ✅ Android
- ✅ iOS
- ✅ Web
- ✅ Desktop (Windows, macOS, Linux)

---

## 🔒 Privacy & Security

### Does Socialize have E2E encryption?

Yes! Socialize implements end-to-end encryption using the Signal protocol. Only you and the recipient can read messages.

### Where is data stored?

You decide! Socialize can be:
- Self-hosted (your own servers)
- Cloud managed (when available)

Your data is always encrypted at rest.

### Does Socialize collect data?

We don't collect unnecessary personal data. See our [Privacy Policy](./security/privacy-policy.md) for details.

### How to report security bugs?

Send an email to security@socialize.app. We'll respond within 24 hours.

## 💬 Messages

### How to create a group?

1. Open Socialize
2. Tap "New Chat"
3. Select "New Group"
4. Add participants
5. Set name and photo
6. Tap "Create"

### How many people can be in a group?

Groups can have up to 1,000 members.

### Can messages be scheduled?

Yes! Long-press the message and select "Schedule". Choose date and time.

---

## ⚙️ Configuration

### How to change theme?

1. Go to Settings
2. Tap "Appearance"
3. Select "Theme"
4. Choose from Light, Dark, AMOLED, or Custom

### Can I create my own theme?

Yes! You can create custom themes using our integrated Theme Editor.

---

## 🐛 Problems

### App won't connect

1. Check your internet
2. Confirm the server is running
3. Clear cache and data
4. Reinstall if necessary

### Messages don't deliver

1. Check your internet
2. Update to the latest version
3. Contact us if it persists

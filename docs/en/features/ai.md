## 🌟 Dandara AI

### What Is It?

AI agent suite with:
- Chat assistant
- Smart replies
- Real-time translation
- Summarization
- Content generation

### Activate

1. Settings → AI
2. Dandara AI
3. Activate

---

## 💬 Chat Assistant

### How to Use

In any conversation:

1. Type `/ai` or tap 🤖
2. Ask your question
3. Dandara responds

### Commands

| Command | Description |
|---------|-----------|
| `/ai [question]` | Direct question |
| `/ai summarize` | Summarize chat |
| `/ai translate [text]` | Translate text |

---

## 💡 Smart Replies

Based on received messages, Dandara suggests quick responses:

```
💡 "How are you?" → "I'm good, thanks!"
💡 "Want to hang out?" → "Can't, I need to..."
```

---

## 🌍 Real-Time Translation

### How to Use

1. Long-press the message
2. Select "Translate"
3. Choose language

### Supported Languages

80+ languages including Portuguese, English, Spanish, French, German, Chinese, Japanese, Korean.

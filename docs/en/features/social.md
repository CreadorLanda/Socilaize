# 🌟 Social Features

> Documentation for Socialize social features.

---

## 📖 Stories

### What Are Stories?

Temporary posts that last 24 hours. Can include:
- Text
- Images
- Videos
- Links

### Create Story

1. Tap "📷" in stories tab
2. Capture or select media
3. Add text/effects
4. Publish

## 🎭 Multi-Identity Profiles

### What Are They?

Different profiles for different audiences:

| Profile | For |
|---------|------|
| 👫 Friends | Close friends |
| 💼 Work | Colleagues |
| 🎮 Gaming | Gaming community |
| 👨‍👩‍👧‍👦 Family | Family |

### Create Profile

1. Profile → + New Profile
2. Select type
3. Customize
4. Configure audience

## 🎵 Music Status

### What Is It?

Shows the song you're listening to:

```
🎵 "Bohemian Rhapsody - Queen"
```

### Activate

1. Settings → Status
2. Music Status
3. Connect Spotify/Apple Music

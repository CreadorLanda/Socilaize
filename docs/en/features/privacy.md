# 🔒 Privacy

> Documentation for Socialize privacy features.

---

## 👻 Ghost Mode

### Overview

Ghost Mode allows using Socialize completely invisible. Others cannot know you're online or have read their messages.

### Activate Ghost Mode

1. Settings
2. Privacy
3. Ghost Mode
4. Activate

### Ghost Levels

| Level | What's Hidden |
|-------|--------------|
| 🔵 Light | Last seen |
| 🟡 Medium | Online + Typing |
| 🔴 Full | Everything = fully invisible |

## 🔐 App Lock

### Activate Lock

1. Settings → Security
2. App Lock
3. Choose method:
   - PIN (4-6 digits)
   - Pattern
   - Biometric
   - Fingerprint

## 🎭 Anti-Delete Messages

### What It Does

When enabled, messages are NOT deleted even when the sender tries to delete them.

### Activate

1. Settings → Privacy
2. Anti-Delete
3. Activate

---

### Supported Biometrics

- Iris

---

## 🔒 End-to-End Encryption (E2E)

### How It Works

1. Each user has a unique private key
2. Messages are encrypted before sending
3. Only recipient can decrypt
4. Server CANNOT read messages

### Verify Security

1. Open conversation
2. Tap the name
3. Encryption
4. Scan contact's QR code

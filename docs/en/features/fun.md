# 🎮 Fun Features (MSN-style)

> Documentation for Socialize fun features inspired by MSN Messenger.

---

## 🔔 Nudges

### What Are Nudges?

A "poke" notification that gets attention:

| Emoji | Name | Message |
|-------|------|---------|
| 👋 | Wave | " is saying hi! 👋" |
| 💨 | Hey | " is showing up! 💨" |
| 😱 | Wow | " is impressed! 😱" |

### Send Nudge

1. Open chat
2. Tap ✉️ → Nudge
3. Select type

---

## 😉 Winks

### What Are Winks?

An interactive wink that appears on the recipient's screen:

| Wink | Animation |
|------|----------|
| 😉 Wink | Eye winking |
| 😘 Kiss | Flying heart |
| 😎 Cool | Sunglasses |
| 😁 Laugh | Laughing |
| 🤔 Think | Thinking |

## 🖼️ Display Pictures (DP)

### What Are DPs?

Profile picture that appears next to your name, like MSN:

### DP Types

- Static image
- Animated GIF
- Short video

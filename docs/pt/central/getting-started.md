# 🚀 Primeiros Passos

> Guia completo para começar com o Socialize.

## Pré-requisitos

| Requisito | Versão |
|----------|--------|
| Node.js | 18.x+ |
| Go | 1.21+ |
| Docker | Latest |

## Início Rápido

```bash
git clone https://github.com/socialize/socialize.git
cd socialize
npm install
npm run dev
```

## Configuração

Crie o arquivo `.env`:

```env
DATABASE_URL=postgresql://user:pass@localhost:5432/socialize
JWT_SECRET=sua_chave_secreta
```

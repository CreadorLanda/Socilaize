# 🚀 Getting Started

> Complete guide to get started with Socialize.

## Prerequisites

| Requirement | Version |
|-------------|---------|
| Node.js | 18.x+ |
| Go | 1.21+ |
| Docker | Latest |

## Quick Start

```bash
git clone https://github.com/socialize/socialize.git
cd socialize
npm install
npm run dev
```

## Configuration

Create `.env` file:

```env
DATABASE_URL=postgresql://user:pass@localhost:5432/socialize
JWT_SECRET=your_secret_key
```
